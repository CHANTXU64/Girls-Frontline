**该如何维护更新后勤战役?**<br>
只需要修改js/GF_logistics_data.js, 修改Q数组和setQContract()函数即可.<br>
(需要注意目前最大能支持到第20战役, 在此之后需要修改lang/languages.js, 增加对应战役的显示语言, 如"chapter21": "第二十一战役")
